# PaloPulse
PaloPulse is a real-time security visualizer for Palo Alto NGFWs. It transforms PAN-OS logs into a live geographic dashboard using Python &amp; Socket.io. Track Layer 3/4 traffic and Layer 7 threats via a sleek "cyber-noir" map with dynamic pulsing markers, active leaderboards, and instant threat intelligence. Engineered for modern SOC integration.
